Steps to make the Tag Simulator work:

1- After installing the TrakSYS on your computer, copy the 3 following files and paste them in the same    directory as you have installed the TrakSYS (e.g. "C:\Program Files\Parsec\TrakSYS"):
   TagSimulator.exe 
   TagSimulator.xml
   TagSimulator.exe.settings.config

2- You can create a shortcut of TagSimulator.exe on your desktop.

3- Run the TagSimulator.exe
